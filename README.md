# Лабораторная работа №4. Массивы и функции

## Инструкции по запуску

1. Скопируйте папку `lab4` в директорию `htdocs` XAMPP.
2. Запустите Apache.
3. Откройте `http://localhost/projects/usm/lab4/index.php`.

Проверка синтаксиса:

```bash
/Applications/XAMPP/xamppfiles/bin/php -l index.php
```

## Описание работы

Реализована система управления банковскими транзакциями на массивах PHP:

- создан массив транзакций с полями `id`, `date`, `amount`, `description`, `merchant`;
- добавление новой транзакции выполняется функцией `addTransaction()`;
- поиск реализован по части описания, по ID через `foreach` и по ID через `array_filter`;
- подсчитывается общая сумма транзакций обычным циклом `foreach`;
- вычисляется количество дней с даты транзакции;
- выполнена сортировка по дате и по сумме по убыванию через `usort()`;
- данные выводятся в HTML-таблицах.

Вторая часть работы посвящена файловой системе. В директории `image` находится 24 JPG-файла с Unsplash на тему финансов, платежей, покупок и учета расходов. Скрипт считывает изображения через `scandir()` и выводит их в виде галереи.

## Краткая документация

- `calculateTotalAmount()` считает общую сумму.
- `findTransactionByDescription()` ищет транзакции по фрагменту описания.
- `findTransactionById()` ищет транзакцию по ID через `foreach`.
- `findTransactionByIdFilter()` ищет транзакцию по ID через `array_filter`.
- `daysSinceTransaction()` возвращает количество дней с даты транзакции.
- `addTransaction()` добавляет запись в глобальный массив `$transactions`.
- `getGalleryImages()` считывает изображения из директории.

## Примеры использования

```php
$total = calculateTotalAmount($transactions);
$transaction = findTransactionById($transactions, 5);
$matches = findTransactionByDescription($transactions, 'course');
```

## Ответы на контрольные вопросы

1. Массивы в PHP - это структура данных для хранения набора значений. Они могут быть индексными, ассоциативными или многомерными.
2. Массив можно создать через короткий синтаксис `[]` или функцию `array()`. Например: `$items = ['a', 'b'];`.
3. `foreach` используется для перебора элементов массива. Он удобен, когда нужно обработать каждый элемент без ручного управления индексом.
